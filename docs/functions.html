KSL Functions

Functions aren't particularly interesting. Mostly just the same old-same old from other languages. Here's a function that returns nothing: 
function main() -> null {
	return;
}
Here's a function that returns an integer: 
function main() -> int {
	return 1;
}
Here's a function that takes an integer and returns it: 
function main(int a) -> int {
	return a;
}
Finally, here's a function that uses the nofail attribute so that it won't crash the program if something inside of it breaks. Think of it like a big try-catch block for the entire function. 
@[nofail]
function api_divide_by_zero(int a) -> int {
	return a / 0;
}
Since you can't divide by zero the function will fail, but because of the nofail attribute, it won't crash your entire program.

For exact details on how the nofail attribute works, please see nofail.
