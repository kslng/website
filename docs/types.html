KSL Types

At the time of writing the KSL compiler is very young and doesn't have much type support. Currently the "supported" types are:
  • int (i64 internally)
  • bool (i1 internally)
  • float (f64 internally)
  • null (nullptr internally)
Although only the int type has been rigorously tested.

You can expect plenty of updates to this page as the compiler matures. Additionally, if you would like to help build the compiler and its type system, please go ahead and start contributing to development!
