# Compiling KSL-STDLIB

To compile the standard library:  
Requirements:

- KSL Source Code
- Clang (for windows)
- GCC (for linux)

Steps:

1. Navigate to the `/std/` folder
2. Run the `build.cmd` (windows) or `build.sh` (linux)
- (Optionally) run `build.cmd debug` to enable debug messages in runtime
  libraries (`build.sh debug` for linux)
